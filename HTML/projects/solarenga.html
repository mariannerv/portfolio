About the project

This project is a web service designed to help users find short vacation trips to European capitals with favorable weather conditions. Users can specify their departure city and maximum flight budget, and the system will return vacation options that meet these criteria. The system ensures only direct flights with at least two days of sunny weather and limits trips to four days. It is built using a RESTful architecture, with a backend powered by Flask and a SQLite database. The project integrates external APIs for weather forecasts and flight information to provide up-to-date and accurate trip suggestions.

Security and efficiency are also enhanced in a later phase, introducing SSL/TLS for secure communicatio and Zookeeper for managing client sessions.
