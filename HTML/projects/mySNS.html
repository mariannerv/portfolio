About the project

This project had us develop a secure distributed file storage system called mySNS, designed for managing medical files like exams and prescriptions. The system ensures confidentiality, integrity, and authenticity using Java's security API and follows best practices in cryptographic protocols.

Key features include:

1. File Encryption and Signing: Implementation of hybrid encryption (AES for file encryption, RSA for key encryption) and digital signatures to protect files during transmission and storage.

2. User Authentication and Access Control: The system authenticates multiple users with hashed passwords (salted) and controls access to files, preventing unauthorized access.

3. TLS-secured Communication: Client-server interactions are protected using TLS to ensure secure data transmission. Advanced Server Security: I configured firewall rules and set up intrusion detection with Snort, ensuring robust server protection.

This project involved tackling real-world security challenges such as handling adversarial threats, file encryption, and managing secure communications in a distributed environment.
