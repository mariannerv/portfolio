About the project

This was a group project where we developed a web application designed to make it easier for citizens to report various issues. These issues could range from potholes in the road to maintenance requests, and even suggestions for improvements. The goal was to facilitate smoother communication between citizens and Lisbon's Town Hall, allowing for better responsiveness in addressing public concerns.

The app was divided into two roles: the regular user and the Town Hall employee. Regular users could report new occurrences and track the status of issues marked as 'Open' or 'In Progress.' All registered users were also able to report new occurrences and provide feedback on any of the 'Solved' issues. Any regular user would also be able to connect via live chat to a town hall employee.

As for the town hall employees, they were able to manage all occurrences, check if there were any open chats and everything a regular user can do.

We were also tasked with implementing an external web service that other companies could use to filter and search for occurrences based on their specialty for extra points.
